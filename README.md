## TaxCalculator

### Tools Used

* Ruby 2.1.5
* Rspec

Run it from command line

``` ruby tax_calculator.rb 'input.csv' 'output.csv' ```

### Sample Input
Quantity, Product, Price


1, book, 12.49

1, music cd, 14.99

1, chocolate bar, 0.85

### Sample Output
1, book, 12.49

1, music cd, 16.49

1, chocolate bar, 0.85

Sales Taxes: 1.50

Total: 29.83
